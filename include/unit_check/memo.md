数: R
普通計算といったときに操作するほう

単位: U = Z × Z × ... × Z
次元計算とかで操作するほう。整数の組。

量: R × U
数と単位の組。

通常の量計算
単位が同じとき、和と差が定義される。そうでないときは定義されない
(1, (1, 0, -1)) + (2, (1, 0, -1)) = (1 + 2, (1, 0, -1) + (1, 0, -1)) = (3, (1, 0, -1))
+がオーバーロードされていることに注意。

(1, (1, 0, -1)) + (2, (2, 3, 1)) (?)

全ての量について積が、割るほうの量の数が0でないとき商が定義される。
(2, (1, 0, -1)) * (3, (2, 3, 1)) = (2 * 3, (1, 0, -1) * (2, 3, 1)) = (6, (3, 3, 0))
(2, (1, 0, -1)) / (3, (2, 3, 1)) = (2 / 3, (1, 0, -1) / (2, 3, 1)) = (2/3, (-1, -3, -2))
